 # Setup and deployment instructions
 ## Requirements
 Before cloning the application, make sure you have installed:
 - NodeJs
 (Here is the link for the installation: [Node.js](https://nodejs.org/en))
 - Npm
  (Here is the link for the installation: [npm](https://www.npmjs.com/))
 - Git
  (Here is the link for the installation: [Git](https://git-scm.com/))

---

 ## Application cloning

```bash
git clone https://github.com/fanomezantsoa-angela/Document_viewer.git
```
## Dependency installation
### Document Viewer dependency installation
```bash
cd Document_viewer
npm i
```
### Dashboard dependency installation
```bash
cd Document_viewer
cd app-Dashboard
npm i
```
## Starting the applications
### Starting the backend
```bash
cd backend
npm run dev 

### Starting the document Viewer application
```bash
cd Document_viewer
npm run dev 
```
### Starting the dashboard
```bash
cd NeuroKnowIa_Test
cd app-Dashboard
npm run dev
```
